# 🎓 Exam Seating Management System

A simple and interactive web application built using **Streamlit** to manage and visualize exam seating arrangements efficiently.

## 🌐 Live Demo

👉 https://exam-seating-system-xnyd46fg3pqsfptadrr2dv.streamlit.app/

---

## 🚀 Features

- 🏫 Create and manage multiple exam halls  
- 📊 Automatic seating arrangement generation  
- 🧑‍🎓 Assign students to specific seats  
- 🗺️ Visual representation of seating layout  
- 🗄️ SQLite database integration for data storage  
- ⚙️ Configurable hall size (rows × columns)

---

## 🛠️ Tech Stack

- **Frontend:** Streamlit  
- **Backend:** Python  
- **Database:** SQLite  
- **Libraries:** Pandas  

---

## 📂 Project Structure
examsystem.py       # Main Streamlit application
exam_seating.db     # SQLite database (auto-created)

---

## ⚡ How to Run Locally

### 1. Clone the repository
git clone https://github.com/your-username/exam-seating-system.git
cd exam-seating-system

### 2. Install dependencies
python3 -m pip install streamlit pandas

### 3. Run the app
streamlit run examsystem.py
